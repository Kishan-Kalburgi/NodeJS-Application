# NodeJS-Note-App

git clone: https://github.com/Kishan-Kalburgi/NodeJS-Note-App.git

npm intall

node app.js

example:  node .\app.js add --title='to buy from store' --body="food"

Commands:
  add     Add a new note
  list    List all notes
  read    Read a note
  remove  Delete Above message

Options:
  --help  Show help                                                    [boolean]
